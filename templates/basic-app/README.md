# n8n App Template

Basic Next.js + Tailwind CSS template for n8n workflow frontends.

## Setup

1. Copy this template to `../apps/[your-app-name]/`
2. Run `npm install`
3. Update the webhook URL in `src/app/page.tsx`
4. Run `npm run dev` to test locally

## Deploy

1. Push to GitHub
2. Connect to Vercel
3. Set environment variables if needed
